# python_kilominx

kilominx simulation in python (pygame)

## \#\# Prerequises \#\#

Python installed on PC

## \#\# Install \#\#

If Linux / MacOS

```sh
./install.sh
```

If Windows

```bat
.\install.bat
```

## \#\# Run \#\#

If Windows / MacOS

```sh
python main.py
```

If Linux

```sh
python3 main.py
```
